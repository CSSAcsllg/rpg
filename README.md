# 使用说明
解压本压缩包，在 *.py* 文件对应的目录下打开终端，依次执行
~~~
pip install numpy
pip install pygame
python rpg.py
~~~
# 玩法说明
 *a* *d* ：左右移动

 *w* 或 *k* ：跳跃

 *j* ：进攻

 *i* ：免疫技能

 在城堡处使用 *q* 进入城堡，选择关卡

 击败一个阶段敌人后，点击右下角按钮可以回到城堡，也可从城堡再次返回

 使用 *n* 进入下一阶段

 主界面在小女孩出使用 *q* 购买生命
